# Roadmap

This roadmap focuses on practical, visible milestones.

## v0.1 Foundation
- Multi-user login
- Per-user runtime containers
- Unified model gateway
- Minimal Web UI
- Minimal admin view
- Private user workspace

## v0.2 Shared Space
- Team shared space model
- Shared space permissions
- Shared space mounting in runtimes
- Basic shared space management UI

## v0.3 Local Model Support
- Better local model integration with vLLM
- Ollama as supplemental local backend
- Admin-managed model availability
- Default routing and model policy improvements

## v0.4 Admin and Governance
- Better admin console
- User lifecycle management
- Runtime inspection and troubleshooting
- Usage visibility and basic audit records
- Quota and policy improvements

## v0.5 Local Folder Connector
- Desktop connector design and first implementation
- User-authorized local folder access
- Read/write permission controls
- Sync or direct-access strategy validation
- Cross-platform support for Windows, Linux, and macOS

## Long-Term
- Stronger governance and auditability
- More deployment options
- Better observability
- More flexible runtime backends
- More polished enterprise controls
